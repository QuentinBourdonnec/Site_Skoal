Créer une nouvelle base de donnée et veuillez y importer le fichier SQL 'db-projetphp.sql'

Changer la configuration de la base de données dans le fichier '/php/bdd.php' :
 - le nom de votre serveur : $servername
 - votre nom d'utilisateur sur mysql ou phpmyadmin : $username
 - votre mot de passe : $ password
 - le nom de la base de données que vous venez de créer : $dbname
